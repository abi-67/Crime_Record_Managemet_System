# Crime Record Management System

## Overview
A desktop application built with Java Swing to manage crime records including criminals, crimes, victims, evidence, court judgements and prison management.

## Technologies Used
- Java (Swing GUI)
- MySQL Database
- JDBC for DB connectivity

## Features
- 17 modules with full CRUD operations
- Dynamic search across all modules
- JOIN queries for relationship tables
- Crime-Criminal, Crime-Victim, Crime-Officer relationship tracking

## Setup
1. Install MySQL and create database: `CrimeRecordManagementSystem`
2. Run the SQL scripts to create tables
3. Update `DBConnection.java` with your MySQL password
4. Open project in IntelliJ IDEA
5. Add MySQL Connector/J JAR to project libraries
6. Run `MainFrame.java`

## Modules
17 modules including Criminals, Crime, Police Station, Officers, Victim, Evidence, Witness, Forensic Report, Weapon Registry, Court, Judgement, Sentence, Prison, Inmates and relationship tables.
